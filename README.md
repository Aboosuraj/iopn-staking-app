# iopn-staking-app
IOPn Staking DApp is a mobile-first decentralized application built on the IOPn Testnet that allows users to stake IOPn, tUSDT, and tBNB tokens, earn rewards, and manage staking positions through a clean Web3 interface.

# ⛓️ IOPn Staking DApp (Testnet)

A modern decentralized staking application built for the **IOPn Testnet**, allowing users to stake multiple tokens, view balances, and earn rewards through a simple, mobile-friendly Web3 interface.

---

## 🚀 Features

- 🔐 Wallet connection (MetaMask, WalletConnect)
- 💰 Stake multiple tokens:
  - IOPn
  - tUSDT
  - tBNB
- 📊 View staking balances
- 🎁 Reward tracking system (UI-ready)
- 📱 Fully mobile responsive design
- ⚡ Fast UI built with Next.js App Router
- 🔗 Web3 integration using Wagmi + Viem
- 🎨 Modern UI with gradient themes and glassmorphism cards

---

## 🧱 Tech Stack

- **Frontend:** Next.js (React App Router)
- **Styling:** TailwindCSS
- **Web3:** Wagmi + Viem
- **Wallets:** RainbowKit
- **State/Data:** React Query
- **Blockchain:** IOPn Testnet
